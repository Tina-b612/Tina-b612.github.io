一些小知识点

  • 1.使用对象进行判断时，对比的是对象在栈中的址，所有多次声明的一模一样的对象对比时，也是不相等的。只声明了一次的对象，赋给不同的变量时，变量相等
  • 2.break（打断循环继续执行）一般是用在switch、for里，只能停止当前这个循环，一般用法是，判断满足某条件时，停止执行
  • 3.continue（跳过当次for循环，继续执行下面的代码）用在for循环里，一般用法是，判断满足某条件时，跳过当次
  • 4.当不使用var声明变量的时候，直接进行赋值，程序会自动创建这个变量，并且默认为全局变量
  • 5.对象是无序集合
  • 6.null == undefined // true
    null === undefined // false
  • 7.innerHTML 后跟 字符串。当后面跟对象时，会自动解析成字符串，类型+构造函数
